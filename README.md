1 警务员通过微信平台可打开系统，用户要想使用本系统，必须进行登录操作，没有账号的警务员可进行注册操作，警务员注册登录后主要功能模块包括首页、公告信息以及我的（任务信息、出勤信息和考勤信息）。
 
2 管理员功能分析：管理员通过微信平台登录系统后，可对系统进行全面管理，包括个人中心、警务员管理、公告信息管理、任务信息管理、出勤信息管理、考勤信息管理、管理员管理以及系统管理。
